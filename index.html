Additional Features (Soccer Only)

Notes: I got a little enthusiastic with building this version, and had some fun adding some of the suggested additional gameplay
features. This demo utilizes some additional features of Rive and it's ability to send / receive from a web app portion.
The scoreboard that was added, is completely built in Rive, including the UI / buttons. Vue / JS portion listens for events fired from
the Rive portion for gameplay resets and continuing through the three attempts.
Vue / JS transmit back to the visual portion additional state updates for; attempt number, score, and toggling displaying elements. These are used
within the Rive state machine for animation handling and transitions. Football scene is disabled in this demo.

  • Three attempts style gameplay
  • Results screen after each attempt showing score progress.
